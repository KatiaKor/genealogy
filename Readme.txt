For matlab version 9.4 (R2018a) or higher

English part:
There are two main functions. It:
Pers_Gen.m for creating relatives
and
Pers_sheme2.m / Pers_sheme.m to create a family tree
In Pers_Gen.m, you can edit the lists of names for the victims of selection.
It is worth noting that there are years of birth of people here, but (since it was originally not quite for people) they are unrealistic, because these "people" live longer than us.
Other functions - supporting and are used in the basic functions.


Русская часть:
Тут есть две основные функции. Это:
Pers_Gen.m для создания родственников
и
Pers_sheme2.m / Pers_sheme.m для создания генеалогического древа
В Pers_Gen.m можно подредактировать списки имен и фамилий для жертв селекции.
Стоит отметить, что тут есть года рождения людей но (так как оно изначально было не совсем для людей) они малореалистичны, ибо эти "люди" живут дольше нас.
Остальные функции - вспомогательные и используются в основных функциях.
